# Agile-workflow
Agile Workflow: Receita de Bolo para Programação 🍰 Este repositório contém o meu guia prático e padronizado para o desenvolvimento de software. A intenção é ter um passo a passo (checklist) que elimina a indecisão e garante que todo projeto siga o mesmo padrão de qualidade, da primeira linha de código ao deploy.
